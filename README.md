# Design Skills for Developers

> A practical UI/UX design guide for developers and vibe coders who want to create better AI design prompts.

## 📚 What's Inside

This repository contains a comprehensive guide to help non-designers understand design fundamentals and write more effective prompts for AI design tools.

### Contents

1. **Design Fundamentals** - UI vs UX, design thinking
2. **UI Design Principles** - Visual hierarchy, consistency, alignment, proximity, contrast
3. **UX Design** - User journey, information architecture, usability, accessibility
4. **Design Systems** - Common components and their states
5. **Color & Typography** - Color theory and type systems
6. **Responsive Design** - Breakpoints and layout strategies
7. **AI Prompt Techniques** - Templates, vocabulary, examples
8. **Design Terms Glossary** - Quick reference for common terminology
