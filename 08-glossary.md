# 08 - 常用设计术语速查

> 中英文对照，快速查找设计术语。

---

## 布局相关

| 中文 | 英文 | 含义 |
|------|------|------|
| 留白 | White Space / Negative Space | 元素之间的空白区域 |
| 边距 | Margin | 元素外部的间距 |
| 内边距 | Padding | 元素内部的间距 |
| 栅格 | Grid | 页面布局的网格系统 |
| 列 | Column | 垂直分割的页面区域 |
| 行 | Row | 水平排列的元素组 |
| 容器 | Container | 包裹内容的盒子 |
| 全宽 | Full-width | 占满整个屏幕宽度 |
| 固定宽度 | Fixed-width | 固定像素宽度 |
| 最大宽度 | Max-width | 元素的最大宽度限制 |

## 视觉样式

| 中文 | 英文 | 含义 |
|------|------|------|
| 圆角 | Border Radius | 边角的弯曲程度 |
| 阴影 | Shadow / Drop Shadow | 元素下方的阴影效果 |
| 渐变 | Gradient | 颜色平滑过渡 |
| 透明度 | Opacity | 元素的透明程度（0-100%） |
| 模糊 | Blur | 模糊效果 |
| 边框 | Border | 元素的边界线 |
| 背景色 | Background Color | 元素的背景颜色 |
| 前景色 | Foreground Color | 元素的前景色（通常是文字） |
| 饱和度 | Saturation | 颜色的鲜艳程度 |
| 明度 | Lightness / Brightness | 颜色的明暗程度 |

## 组件类型

### 导航

| 中文 | 英文 | 含义 |
|------|------|------|
| 导航栏 | Navbar / Navigation Bar | 顶部导航 |
| 侧边栏 | Sidebar | 侧边导航 |
| 标签页 | Tabs | 切换内容的标签 |
| 面包屑 | Breadcrumb | 显示当前位置的导航 |
| 分页 | Pagination | 内容分页导航 |
| 下拉菜单 | Dropdown / Select | 点击展开的菜单 |
| 汉堡菜单 | Hamburger Menu | 三条横线的菜单图标 |
| 工具提示 | Tooltip | 悬停显示的小提示 |

### 输入

| 中文 | 英文 | 含义 |
|------|------|------|
| 输入框 | Input / Text Field | 单行文字输入 |
| 文本域 | Textarea | 多行文字输入 |
| 下拉选择 | Select / Dropdown | 从选项中选择 |
| 单选框 | Radio Button | 多选一 |
| 复选框 | Checkbox | 多选多 |
| 开关 | Switch / Toggle | 二选一（开/关） |
| 滑块 | Slider | 拖动选择数值 |
| 日期选择器 | Date Picker | 选择日期 |
| 文件上传 | File Upload | 上传文件 |
| 搜索框 | Search Box | 搜索输入 |

### 展示

| 中文 | 英文 | 含义 |
|------|------|------|
| 卡片 | Card | 信息分组展示 |
| 列表 | List | 垂直排列的信息 |
| 表格 | Table | 结构化数据展示 |
| 折叠面板 | Accordion | 可展开收起的内容 |
| 标签 | Tag / Badge | 状态或分类标识 |
| 头像 | Avatar | 用户头像展示 |
| 徽章 | Badge | 数字或状态提示 |
| 轮播图 | Carousel / Slider | 滑动展示的图片 |
| 模态框 | Modal / Dialog | 弹出式对话框 |
| 骨架屏 | Skeleton | 加载时的占位动画 |

### 反馈

| 中文 | 英文 | 含义 |
|------|------|------|
| 提示消息 | Toast / Notification | 短暂显示的操作反馈 |
| 警告框 | Alert | 警告信息提示 |
| 确认框 | Confirm Dialog | 需要确认的对话框 |
| 进度条 | Progress Bar | 显示完成进度 |
| 加载动画 | Spinner / Loader | 加载状态动画 |
| 空状态 | Empty State | 无数据时的提示 |
| 错误页面 | Error Page | 404 或其他错误 |

## 设计原则

| 中文 | 英文 | 含义 |
|------|------|------|
| 视觉层次 | Visual Hierarchy | 引导视线的层级关系 |
| 一致性 | Consistency | 相同功能相同表现 |
| 对齐 | Alignment | 元素的位置关系 |
| 亲密性 | Proximity | 相关元素靠近 |
| 对比 | Contrast | 元素之间的差异 |
| 平衡 | Balance | 视觉重量的均衡 |
| 节奏 | Rhythm | 重复产生的规律感 |
| 统一 | Unity | 整体的和谐感 |

## 用户体验

| 中文 | 英文 | 含义 |
|------|------|------|
| 用户旅程 | User Journey | 用户完成任务的流程 |
| 用户流程 | User Flow | 用户在应用中的路径 |
| 信息架构 | Information Architecture | 信息的组织结构 |
| 可用性 | Usability | 易用程度 |
| 无障碍 | Accessibility / A11y | 残障人士可用性 |
| 交互设计 | Interaction Design | 用户与产品的互动 |
| 用户研究 | User Research | 了解用户的方法 |
| 原型 | Prototype | 可交互的模型 |
| 线框图 | Wireframe | 低保真设计稿 |
|  mockup | 效果图 | 高保真设计稿 |

## 技术术语

| 中文 | 英文 | 含义 |
|------|------|------|
| 响应式 | Responsive | 适配不同屏幕 |
| 自适应 | Adaptive | 多套固定布局 |
| 断点 | Breakpoint | 响应式布局的临界点 |
| 移动端优先 | Mobile First | 先设计手机版 |
| 桌面端优先 | Desktop First | 先设计桌面版 |
| 像素 | Pixel / px | 屏幕上的点 |
| 视口 | Viewport | 浏览器可视区域 |
| 分辨率 | Resolution | 屏幕像素数量 |
| DPI | Dots Per Inch | 每英寸点数 |
| 矢量 | Vector | 可无限缩放的图形 |
| 位图 | Bitmap / Raster | 像素组成的图像 |
