# 05 - 配色与排版

> 让界面既好看又易读。

---

## 配色基础

### 色彩模式

| 模式 | 格式 | 使用场景 |
|------|------|---------|
| **HEX** | `#FF5733` | 网页开发最常用 |
| **RGB** | `rgb(255, 87, 51)` | CSS、设计软件 |
| **HSL** | `hsl(9, 100%, 60%)` | 调整颜色最直观 |

**HSL 解释**：
- **H**ue（色相）：0-360 度的色轮位置
- **S**aturation（饱和度）：0% 灰色 → 100% 纯色
- **L**ightness（明度）：0% 黑色 → 50% 纯色 → 100% 白色

## 配色方案类型

### 1. 单色 (Monochromatic)

**特点**：同一色相的不同明度

**优点**：
- 安全、专业
- 简洁、统一
- 易于实现

**示例**：
```
深蓝 #1a365d（主色）
蓝色 #3182ce（辅色）
浅蓝 #bee3f8（背景）
```

**适用场景**：企业网站、工具类应用

---

### 2. 类比色 (Analogous)

**特点**：色轮上相邻的颜色（间隔 30°）

**优点**：
- 和谐、自然
- 有层次但不冲突

**示例**：
```
蓝色 #3182ce
蓝绿 #38b2ac
绿色 #48bb78
```

**适用场景**：自然主题、健康类应用

---

### 3. 互补色 (Complementary)

**特点**：色轮上相对的颜色（间隔 180°）

**优点**：
- 对比强烈、醒目
- 突出重点

**示例**：
```
蓝色 #3182ce
橙色 #ed8936
```

**适用场景**：CTA 按钮、警告提示

---

### 4. 三色 (Triadic)

**特点**：色轮上等距的三种颜色（间隔 120°）

**优点**：
- 丰富、平衡
- 活泼有趣

**示例**：
```
蓝色 #3182ce
红色 #e53e3e
黄色 #ecc94b
```

**适用场景**：儿童产品、创意类应用

---

## 色彩角色

一个完整的设计系统通常包含以下颜色：

| 角色 | 用途 | 示例 |
|------|------|------|
| **主色 (Primary)** | 品牌主色，主要操作按钮 | 蓝色 `#3182ce` |
| **辅色 (Secondary)** | 次要操作，辅助元素 | 灰色 `#718096` |
| **强调色 (Accent)** | 突出特殊元素 | 橙色 `#ed8936` |
| **背景色 (Background)** | 页面背景 | 浅灰 `#f7fafc` |
| **表面色 (Surface)** | 卡片、面板背景 | 白色 `#ffffff` |
| **文字色 (Text)** | 正文和标题 | 深灰 `#1a202c` |
| **错误色 (Error)** | 错误提示 | 红色 `#e53e3e` |
| **成功色 (Success)** | 成功提示 | 绿色 `#48bb78` |
| **警告色 (Warning)** | 警告提示 | 黄色 `#ecc94b` |
| **信息色 (Info)** | 信息提示 | 蓝色 `#3182ce` |

### 颜色使用比例

```
60% 中性色（背景、文字）
30% 主色（主要元素）
10% 强调色（CTA、高亮）
```

## 对比度与无障碍

### WCAG 对比度标准

| 级别 | 正常文字 | 大文字 |
|------|---------|--------|
| **AA（最低）** | 4.5:1 | 3:1 |
| **AAA（最佳）** | 7:1 | 4.5:1 |

### 检查工具

- **WebAIM Contrast Checker** (webaim.org/resources/contrastchecker)
- **Stark** (Figma/Sketch 插件)
- ** axe DevTools** (浏览器插件)

## 排版基础

### 字体分类

| 类型 | 特点 | 示例 | 适用场景 |
|------|------|------|---------|
| **无衬线 (Sans-serif)** | 现代、简洁、屏幕友好 | Inter, Roboto, Helvetica | 大多数界面 |
| **衬线 (Serif)** | 传统、正式、印刷易读 | Merriweather, Georgia | 长文阅读 |
| **等宽 (Monospace)** | 字符等宽 | Fira Code, JetBrains Mono | 代码显示 |

### 推荐字体组合

**组合 1：现代简洁**
```
标题：Inter Bold
正文：Inter Regular
代码：JetBrains Mono
```

**组合 2：专业可信**
```
标题：Merriweather Bold
正文：Open Sans Regular
```

**组合 3：活泼友好**
```
标题：Poppins SemiBold
正文：Nunito Regular
```

## 字体大小层级

### 标准层级

| 级别 | 大小 | 用途 | 字重 |
|------|------|------|------|
| **H1** | 36-48px | 页面主标题 | 700 (Bold) |
| **H2** | 28-32px | 章节标题 | 600 (Semi-bold) |
| **H3** | 22-24px | 小节标题 | 600 (Semi-bold) |
| **H4** | 18-20px | 卡片标题 | 500 (Medium) |
| **Body** | 16px | 正文内容 | 400 (Regular) |
| **Small** | 14px | 辅助文字 | 400 (Regular) |
| **Caption** | 12px | 标签、注释 | 400 (Regular) |

### 缩放比例

常用比例：
- **1.25（Major Third）**：16, 20, 25, 31, 39...
- **1.414（Augmented Fourth）**：16, 22.6, 32, 45...

---

## 行高与字重

### 行高 (Line Height)

| 用途 | 推荐行高 | 说明 |
|------|---------|------|
| **标题** | 1.2 - 1.3 | 紧凑，有力量感 |
| **正文** | 1.5 - 1.6 | 舒适阅读 |
| **大段文字** | 1.6 - 1.8 | 长文阅读更轻松 |

### 字重 (Font Weight)

| 数值 | 名称 | 用途 |
|------|------|------|
| 400 | Regular | 正文 |
| 500 | Medium | 强调文字 |
| 600 | Semi-bold | 小标题 |
| 700 | Bold | 大标题 |

## 实用配色方案

### 方案 1：专业蓝
```css
--primary: #3182ce;
--secondary: #718096;
--background: #f7fafc;
--surface: #ffffff;
--text: #1a202c;
--error: #e53e3e;
--success: #48bb78;
```

```css
--primary: #805ad5;
--secondary: #a0aec0;
--background: #faf5ff;
--surface: #ffffff;
--text: #1a202c;
--error: #f56565;
--success: #48bb78;
```

```css
--primary: #ed8936;
--secondary: #718096;
--background: #fffaf0;

```
