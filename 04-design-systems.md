# 04 - 设计系统与组件

> 了解常见 UI 组件及其使用场景。

---

## 导航组件

### 顶部导航栏 (Navbar)

**用途**：主导航，通常固定在页面顶部

**包含元素**：
- Logo（点击返回首页）
- 主导航链接
- 搜索框
- 用户头像/登录按钮

**Prompt 关键词**：
```
"顶部固定导航栏"
"包含 Logo 和主导航"
"右侧用户菜单"
"滚动时保持可见"
```

---

### 侧边栏 (Sidebar)

**用途**：次级导航，适合工具类、仪表盘类应用

**类型**：
- **固定侧边栏**：始终可见
- **可折叠侧边栏**：点击展开/收起
- **悬浮侧边栏**：鼠标悬停显示

**Prompt 关键词**：
```
"左侧固定侧边栏导航"
"可折叠的菜单结构"
"图标+文字的标签"
"当前页面高亮显示"
```

---

### 标签页 (Tabs)

**用途**：同一层级的内容切换

**适用场景**：
- 商品详情（描述/规格/评价）
- 设置页面（个人/通知/安全）

**Prompt 关键词**：
```
"标签页切换不同内容"
"当前标签高亮显示"
"滑动切换动画"
```

---

### 面包屑 (Breadcrumb)

**用途**：显示当前位置和返回路径

**格式**：
```
首页 > 分类 > 子分类 > 当前页面
```

**Prompt 关键词**：
```
"面包屑导航显示路径"
"可点击返回上级"
"简洁的路径展示"
```

---

### 分页器 (Pagination)

**用途**：大量内容的分页展示

**类型**：
- 传统分页（1 2 3 ... 10）
- 加载更多按钮
- 无限滚动

**Prompt 关键词**：
```
"底部分页导航"
"显示当前页码"
"可跳转指定页面"
```

---

## 组件状态

每个交互组件通常有多种状态：

| 状态 | 说明 | 视觉表现 |
|------|------|---------|
| **默认 (Default)** | 正常状态 | 标准样式 |
| **悬停 (Hover)** | 鼠标移上去 | 颜色变深/背景变化 |
| **聚焦 (Focus)** | 键盘选中 | 外边框/阴影 |
| **激活 (Active)** | 点击中 | 颜色更深/下沉效果 |
| **禁用 (Disabled)** | 不可用 | 降低透明度/灰色 |
| **加载 (Loading)** | 处理中 | 显示加载动画 |
| **错误 (Error)** | 出错状态 | 红色边框/提示 |

**Prompt 关键词**：
```
"按钮包含悬停和点击状态"
"输入框有聚焦状态样式"
"禁用状态降低透明度"
"加载状态显示动画"
```
